## Description

This repository shows how to use nest fundamental functions.

## What we will make

- [x] CRUD
- [ ] DataBase
- [ ] OAuth and Auth
- [ ] Messaging Queue

## Installation

```bash
$ git clone https://github.com/Tocktock/nest-practice
$ cd nest-practice
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
